David is the principal consultant for Cobb Information Technologies since 1996 and a Microsoft Certified Trainer since 2002. David provides training and consulting on database and cloud migration and automation in Azure and AWS using PowerShell and Command Line tools.

He is a CloudSkills.io author, volunteer and speaker at local South Florida user groups and events focused on Azure and the Microsoft stack like Code Camp and SQL Saturday. He has served as a technical reviewer for books including ‘SQL Server 2014 with Powershell v5’ and ‘Pro PowerShell for Microsoft Azure’.
